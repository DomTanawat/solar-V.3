ระบบสุริยะก่อตัวเมื่อประมาณ 4.6 พันล้านปีก่อน จากการยุบตัวของกลุ่มเมฆแก๊สและฝุ่นขนาดยักษ์ (เนบิวลาสุริยะ) แรงโน้มถ่วงทำให้กลุ่มเมฆนี้ยุบตัวลงและหมุนเร็วขึ้น จนกลายเป็นแผ่นจานและใจกลางที่ร้อนจัด ซึ่งพัฒนาจนกลายเป็นดวงอาทิตย์ สสารที่เหลือในจานได้จับตัวกันเป็นก้อนและรวมตัวกันเป็นดาวเคราะห์ ดาวเคราะห์น้อย และดวงจันทร์
